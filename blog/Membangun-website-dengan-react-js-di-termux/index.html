Membangun Website dengan ReactJS di Termux

Pendahuluan

Apakah kalian ingin membuat website tapi tidak memiliki laptop atau komputer? Tenang, kali ini saya akan memberikan tutorial cara membuat website menggunakan ReactJS langsung di HP dengan Termux.

Dengan cara ini, kalian bisa ngoding kapan saja tanpa perlu membawa laptop. Artikel ini cocok untuk pemula yang ingin belajar ReactJS, tapi belum punya perangkat selain HP. Saya akan menjelaskan semuanya secara detail agar mudah dipahami, bahkan oleh pemula sekalipun.

1. Persiapan Awal

1.1 Install Termux

Download Termux dari F-Droid untuk mendapatkan versi yang lebih stabil.

Setelah terinstall, buka Termux dan jalankan perintah berikut untuk memperbarui paket sistem:

Bash
copy
pkg update && pkg upgrade

Jika muncul opsi (Y/n), ketik Y lalu tekan Enter.

1.2 Install Node.js dan npm

ReactJS memerlukan Node.js untuk berjalan. Install dengan perintah berikut:

Bash
copy
pkg install nodejs -y

Setelah instalasi selesai, cek apakah Node.js dan npm sudah terpasang dengan mengetikkan:

Bash
copy

node -v
npm -v

Jika muncul versi Node.js dan npm, berarti instalasi berhasil.

2. Membuat Proyek ReactJS di Termux

Sekarang kita akan membuat proyek ReactJS menggunakan Create React App. Jalankan perintah berikut:

Bash
copy
npx create-react-app my-app

Tunggu hingga proses selesai. Setelah itu, masuk ke folder proyek dengan perintah:

Bash
copy
cd my-app

Lalu, install semua dependensi yang dibutuhkan:

Bash
copy
npm install

3. Menjalankan Proyek ReactJS

Setelah semua siap, jalankan aplikasi React dengan:

Bash
copy
npm start

Jika berjalan dengan lancar, kalian akan melihat output di Termux yang menunjukkan bahwa server berjalan di http://localhost:3000/.

Untuk melihat hasilnya di HP kalian, gunakan browser dan buka:
http://localhost:3000/

4. Mengedit dan Mengembangkan Proyek

Untuk mengedit kode React, kalian bisa menggunakan:

  • VIM di Termux (butuh sedikit belajar dasar VIM).
  • Acode (rekomendasi buat pemula karena lebih user-friendly).

Kalau pakai Acode, tinggal buka folder my-app lalu edit file React di dalamnya seperti src/App.js.

5. Build dan Deploy ke Hosting

Setelah aplikasi selesai dikembangkan, kita bisa build proyek agar siap dihosting. Jalankan:

Bash
copy
npm run build

Proses ini akan menghasilkan folder build/ yang siap di-deploy.

5.1 Deploy ke Netlify

  1. Buat akun di Netlify
  2. Upload folder build/ ke Netlify
  3. Selesai, website React kalian sudah online!

FAQ (Pertanyaan Umum)

1. Apa bisa install library lain di Termux?

Bisa, kalian bisa install library tambahan seperti:

Bash
copy
npm install axios react-router-dom

2. Bisa deploy ke GitHub Pages?

Bisa! Cukup install gh-pages:

Bash
copy
npm install gh-pages -D

Lalu tambahkan script di package.json, kemudian jalankan:

Bash
copy
npm run deploy
